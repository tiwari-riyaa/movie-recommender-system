# movie-recommender-system
# Movie Recommendation System

A content-based movie recommender system built using Machine Learning and Flask.

# Tech Stack
Python
Flask
Pandas
Scikit-learn
TMDB API
# Features
Recommend similar movies
Real-time poster fetching using TMDB API
Web interface using Flask
# How to Run
pip install -r requirements.txt

python app.py


Then open:

http://127.0.0.1:5000/


#dataset
Due to GitHub file size limitations, the tmdb_5000_credits.csv file is not included in this repository.

You can download it from Kaggle:

-> https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

After downloading:

Extract the ZIP file
Place tmdb_5000_credits.csv in the project folder
# Note

This project requires both:

tmdb_5000_movies.csv
tmdb_5000_credits.csv
to run correctly.
